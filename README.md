# Cute

Simplest graphics engine.

# Usage

Build with CMake. Run, choose scenes, use UI to control the stuff.

# Prerequisites

Windows x64.
CMake 3.19 (might be 3.15)
The last MSVC compiler. OpenGL SDK somewhere at default search paths.
Hardware capable of OpenGL 3.3 :)
